Posted on 2021-05-05 In rust基础知识学习

生成线程

  • thread::spwan
  • 结合闭包进行使用，闭包最好带有move关键字，保证所有权的移动
  • 利用join方法，可以将一个线程个另一个线程进行同步化，当且仅当同步化后的线程全部完成，才会进行后续内容

两种数据传递的方式

  • message send
    • 概念：利用管道，进行信息传递
    • mpsc::channel()作为通讯管道
    • 使用通讯管道的发送方（send），和接受方(recv)的方法进行发送或是接受
  • mutex
    • 概念：利用线程锁，进行数据的访问限制
    • mutex自带类似信息量的变量，可以限制访问，另外，mutex内的数据具有mutablity
    • Arc具有原子性，代替Rc(非原子性)，在多线程中进行数据引用
