1099 Build A Binary Search Tree题解

Posted on 2021-03-11

题目链接

大意

给定一颗二叉搜索树的树型，以及一组数据，要求将这一组数据填充到该树型中，在层次遍历这棵树

思路

  1. 将数据从小到大排序
  2. 通过先序遍历（递归型），依次将数据填入即可
    • 可能有人会问这里为什么按先序遍历填入就可以？
    • 首先，我们要认识到二叉搜索树的特点，左孩子比根节点小，右孩子比根节点大
    • 所以，通过先序遍历，依次将排序好的数组填入，刚好就可以满足要求
  3. 最后，通过bfs，层次遍历即可
